## Technology
Required Node.js Installed on Device

- Nest.js
- MongoDB
- JWT
- RabbitMQ

## Install

Install all package using NPM
```bash
npm i
```

## Usage

Docker
```bash
docker-compose up -d --build
docker-compose up
```

Local
```bash
npm run start:dev
```
